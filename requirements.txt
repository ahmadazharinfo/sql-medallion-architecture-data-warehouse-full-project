# Requirements

This project is a SQL Server based data warehouse and does not depend on any
programming language packages. The requirements below describe the tools and
environment needed to run it.

## Database Engine
- Microsoft SQL Server 2019 or later (Developer or Express edition is
  sufficient for local use)

## Client Tools
- SQL Server Management Studio (SSMS) 19 or later

## Diagramming (optional, for docs/*.drawio files)
- draw.io desktop app, or the draw.io VS Code extension

## Version Control
- Git

## Permissions
- The SQL Server service account must have read access to the folder
  containing the source CSV files referenced in
  scripts/bronze/proc_load_bronze.sql (BULK INSERT requires server-side
  file system access, not just client-side access).
